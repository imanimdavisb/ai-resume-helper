# AI Resume Helper

A beginner Python project that helps create resume bullet points based on a job title and work experience.

## Why I Built This

I built this project because I have spent a lot of time updating my resume and applying for jobs. I wanted to create a simple tool that helps generate professional resume bullet points.

## Features

- Accepts a job title
- Accepts work experience
- Generates resume bullet points
- Beginner-friendly Python code

## Skills Used

- Python
- User Input
- String Formatting
- Problem Solving
